README

This project implements a contact management application using object-oriented design principles and multiple design patterns across a series of incremental use cases. The system allows users to register and authenticate, manage their profile, and create, view, edit, delete, and organize contacts. Contacts support multiple phone numbers, email addresses, and tags, and can be searched, filtered, and processed through bulk operations.

The application demonstrates several design patterns including Builder, Factory, Strategy, Command, Memento, Decorator, Observer, Composite, Specification, and Flyweight. These patterns are applied to different features such as user creation, authentication, profile updates, contact creation, viewing formatted contact details, editing with undo support, deletion notifications, bulk processing, search pipelines, advanced filtering, and tag management.

The project also showcases core Java concepts such as collections, sets, streams, lambda expressions, comparators, UUID generation, date and time handling, regex-based searching, and validation. Each use case introduces a specific architectural concept while maintaining a modular, extensible, and well-structured codebase suitable for demonstrating object-oriented design and pattern usage in a contact management system.
